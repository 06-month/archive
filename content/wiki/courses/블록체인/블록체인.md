# 블록체인 (과목 허브)

전공 **블록체인** 과목 MOC. 비트코인 동작을 **암호 기반 → 직렬화 → 트랜잭션 → 생태계** 순으로 쌓는다.

```
ECC → ECDSA 서명 → 트랜잭션(소유권 증명) → 직렬화 → 2.0/알트코인/스마트계약/지갑·채굴
```

- [[타원곡선암호-ECC]] — 유한체 곡선·스칼라곱셈·이산로그·`secp256k1`
- [[ECDSA-디지털서명]] — `(r,s)` 생성·검증, k 재사용 위험
- [[직렬화-SEC-DER]] — SEC·DER·Base58·WIF·엔디언
- [[비트코인-트랜잭션]] — 입력/출력/록타임, UTXO, varint
- [[Script]] — 스택 기반 비튜링완전 잠금/해제 스크립트
- [[블록체인2.0-알트코인]] — 알트코인·합의(PoW/PoS/PoI)
